Adding Styles - Inline vs External

Styles can be declared in an external stylesheet, can be declared in the head tag of same html file, or can be applied directly on the elements inline. This paragraph is using style declared in the head of this page.

Common Selectors

Selector Description Function
h1, p, ... Element/Type Selector Selects all elements with same tag
# ID Selector Select one element with specified id
. Class Selector Select all elements that have the mentioned class
* Universal Selector Selects everything on the page

Cascading and Specificity

inline style > id > class > tag

!important keyword becomes the most specific style and gets precedence over everything

Inheritance

Some properties are inherited from the parent element, while others aren't. Font style is an example of property that is inherited.
